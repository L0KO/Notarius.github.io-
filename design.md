# Дизайн-концепция сайта-визитки нотариуса

## Философия дизайна
Профессиональный, надежный и современный внешний вид, который вызывает доверие и демонстрирует компетентность. Минималистичный подход с акцентом на читаемость и удобство навигации.

## Цветовая палитра
- **Основной цвет**: #2C3E50 (темно-синий) - для заголовков и важных элементов
- **Вторичный цвет**: #34495E (серо-синий) - для подзаголовков и второстепенной информации
- **Акцентный цвет**: #3498DB (светло-синий) - для кнопок и интерактивных элементов
- **Фоновый цвет**: #F8F9FA (светло-серый) - для основного фона
- **Цвет текста**: #2C3E50 (темно-синий) - для основного текста
- **Вспомогательный цвет**: #7F8C8D (серый) - для дополнительной информации

## Типографика
- **Заголовки**: Playfair Display (serif) - элегантный и профессиональный
- **Основной текст**: Inter (sans-serif) - современный и читаемый
- **Размеры**: 
  - H1: 48px
  - H2: 36px
  - H3: 24px
  - Body: 16px
  - Small: 14px

## Визуальные эффекты
- Плавные анимации при прокрутке
- Легкие тени для создания глубины
- Градиенты в акцентных элементах
- Иконки в стиле outline
- Плавные переходы между состояниями

## Компоненты
- Круглые кнопки с тенями
- Карточки услуг с hover-эффектами
- Форма с современными полями ввода
- Интерактивная навигация
- Анимированные счетчики для статистики
